# InfoCocoon (信息茧房)

**Web Filter** 是一款注重隐私的 Chrome 浏览器扩展，旨在通过自动隐藏包含您自定义关键词的帖子、视频和卡片，为您提供更加清爽的浏览体验。

## 功能特性

*   **智能内容屏蔽**: 自动识别并隐藏包含指定关键词的推文、视频或内容卡片。
*   **多平台支持**: 专为以下平台优化：
    *   **Twitter / X** (推文)
    *   **YouTube** (视频推荐、搜索结果)
    *   **Bilibili** (视频卡片、动态流)
*   **隐私优先**: 所有数据（关键词配置）仅存储在您的本地浏览器中（通过 Chrome Storage 同步），绝不上传到任何远程服务器。
*   **高性能**: 使用高效的 DOM 监听技术 (`MutationObserver`) 和防抖处理，确保页面滚动流畅无卡顿。
*   **高度定制**: 支持按站点开启/关闭过滤功能，并提供便捷的关键词管理界面。

## 安装指南

由于本扩展目前处于开发/个人使用阶段，您需要通过“加载已解压的扩展程序”来安装。

1.  克隆或下载本项目代码到您的电脑。
2.  在 Chrome 中打开 `chrome://extensions`。
3.  开启右上角的 **开发者模式 (Developer mode)** 开关。
4.  点击左上角的 **加载已解压的扩展程序 (Load unpacked)** 按钮。
5.  选择本项目的根目录 `web-filter` (包含 `manifest.json` 的文件夹)。

## 使用说明

1.  **添加关键词**:
    *   点击浏览器工具栏上的扩展图标。
    *   在输入框中输入关键词（例如 "剧透", "政治"），然后点击 **添加 (Add)**。
2.  **管理站点开关**:
    *   在弹出窗口中，使用开关按钮来 启用/禁用 当前网站的过滤功能。
    *   支持的站点 (X, YouTube, Bilibili) 默认已启用。
3.  **高级管理**:
    *   点击弹出窗口中的 **齿轮图标** (或右键点击扩展图标 -> 选项) 进入设置页面。
    *   在这里您可以查看所有关键词标签，点击 `×` 移除不需要的关键词，并查看所有站点的启用状态。

## 开发相关

本项目遵循 **公理设计 (Axiomatic Design)** 原则，采用高内聚、低耦合的模块化架构。

*   **代码结构**:
    *   `src/content/`: 内容脚本与特定站点的适配策略。
    *   `src/utils/`: 核心逻辑 (匹配器 Matcher, 配置管理 ConfigManager)。
    *   `src/popup/` & `src/options/`: UI 界面组件。
